welcome to the psh help, this is a list of all commands:
arguments: [optional argument] {required argument}
argument types: s = string n = number b = boolean

    help                Shows help
    exit n[exit_code]   Exits the shell or script with the exit code provided (default: 0)
    version             Displays the version of the application
    cd s{path}          Changes the current working directory to the path provided
    ls                  Lists the items in the current directory
